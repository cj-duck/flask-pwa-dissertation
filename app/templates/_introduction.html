Introduction

Welcome to my development process for building Progressive Web Applications with Flask. This site will guide you through the process of creating a Flask PWA that will preserve as much native-like functionality as possible.

You could follow this process with a non-flask project, making alterations to the examples provided to work with your chosen framework. For many frameworks, however, detailed tutorials are available for creating PWAs.

What is a Progressive Web Application?

If you're looking to develop a cross-platform application, but do not want to develop multiple native applications or use a cross-platform development tool such as Xamarin, PWAs are a web-based alternative. However, the user experience for a web application is comparitively worse than an application developed natively as the user is interacting with the application via a web browser. A PWA can offer functionality previously only available to native applications.

Progressive Web Application (PWA) is a term coined by Google to describe a type of web application that is of sufficient quality to be installed onto a device which, for mobile devices, means adding the application to the homescreen as if it were a native application.

Prerequisite Knowledge

If you follow this process it is assumed that you already know the basics of Flask and can get a basic web application up and running to follow the development process below with. Alternatively if you have a pre-existing Flask application that you wish to turn into a PWA this process will also contain the necessary information to do so, but note your project and file structure may differ from examples used.

Don't know how to use Flask yet? Flask is an incredibly well documented framework with an abundance of tutorials for becoming familiar with how it works. Below are some that I recommend, you'll need to install Python first:

  • The Flask Mega-Tutorial - An extensive tutorial that covers the basics and then dives into more complicated functionality. It also covers deployment which is out of the scope for this development process.
  • Official Flask Tutorial - A tutorial written by the creators of Flask. Less in depth than the mega-tutorial and therefore takes less time to complete.
