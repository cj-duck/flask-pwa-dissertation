HTTPS

For chrome to recognise a site as a PWA and to show the "Add to home screen" prompt one of the conditions is that the site is served over HTTPS. For development, if using the flask development server on localhost, Chrome will still allow the use of features that would otherwise require HTTPS.

When deploying your application you should find an alternative to the flask development server such as Apache and configure the server to handle HTTPS requests and to redirect HTTP requests to HTTPS if possible. This site is hosted with Amazon Web Services, I found this tutorial helpful for setting Flask up with Apache. Let's encrypt was used for obtaining and implementing an SSL certificate.
